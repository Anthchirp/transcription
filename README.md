# transcription
python script to aid in transcribing audio recordings


Python 2
--------
Install with:

  pip install --upgrade git+git://github.com/Anthchirp/transcription.git

Run with:

  transcribe --help

Python 3
--------
Install with:

  pip3 install --upgrade git+git://github.com/Anthchirp/transcription.git

Run with:

  transcribe --help
